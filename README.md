# 📖 About

Scholar AI Researcher is an AI-powered web application that helps students, researchers, developers, and educators understand scientific research papers quickly.

Instead of reading an entire paper manually, users simply upload a PDF and the AI automatically extracts valuable information such as:

- 📄 Summary
- 🔬 Methodology
- 💡 Key Findings
- 📚 Important Insights
- ❓ Question Answering

The goal is to save hours of reading and improve research productivity.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/mayank-kr-coder/Scholar-AI-Researcher.git
```

Go inside folder

```bash
cd Scholar-AI-Researcher
```

Install dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```
